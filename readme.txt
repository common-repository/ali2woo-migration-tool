Ali2Woo Migration Tool allows you to convert products imported by third-party plugins to Ali2Woo format.

== Description ==

Sometimes dropshippers migrate their products from one dropshipping tool to another. It becomes a complex and time consuming process
when it comes to the stores with hundreds and even thouthends products. Fortunelly, our converter addon automate that process allowing you to migrate all imported products and its variants to Ali2Woo format.

[Ali2Woo official website](https://ali2woo.com/) | [Ali2Woo Migration Tool Support](https://support.ali2woo.com/)

### Important Notice:

- Plugin is the addon for the Ali2Woo plugin

- You should have the Ali2Woo plugin installed to work with the addon

- Currently it allows to convert products imported by the following thrird-party dropshipping tools: AliDropship, AliDropship Woocommerce 

- It is released on WordPress.org and you can use plugin as free to build themes for sale.

### Migation Process

After installation and activation of the plugin, you should see the "Migration Tool" in Ali2Woo plugin menu. 

You have to choose your past dropshipping tool from the drop-dawn list on the page, then click "Load products data" to start the process.

It will check your database and convert all products if any exists. There is a progressbar showing the migration process details.

**We highly suggest you to make a database dump before start the migration. It will allow you to rollback changes if something go wrong.**

== Installation ==

= From within WordPress =

1. Visit 'Plugins > Add New'
2. Search for 'Ali2Woo Migration Tool'
3. Activate Ali2Woo Migration Tool from your Plugins page.
4. Go to "after activation" below.

= Manually =

1. Upload the `ali2woo-converter` folder to the `/wp-content/plugins/` directory
2. Activate the Yoast SEO plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
